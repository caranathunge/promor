# Cox et al 2014 LFQ data (normalized)

A dataframe containing normalized LFQ protein intensity data for 4360
proteins in 6 samples

## Usage

``` r
data(ecoli_norm_df)
```

## Format

A data frame with 4360 rows (proteins) and 6 columns (samples)
